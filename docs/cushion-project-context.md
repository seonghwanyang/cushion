# Cushion 프로젝트 컨텍스트

## 🎯 비즈니스 본질

### 핵심 가치 제안
> "실패는 없다, 공백기도 없다. 모든 경험이 당신의 자산이다."

Cushion은 전환기를 겪는 사람들이 자신의 경험을 가치 있는 커리어 자산으로 변환하도록 돕는 AI 일기 서비스입니다.

### 해결하는 문제
1. **공백기 낙인**: 이직 준비, 창업 실패를 부정적으로 보는 사회적 시선
2. **자기 의심**: "나는 뭘 한 사람이지?"라는 불안과 자존감 하락
3. **표현의 어려움**: 전환기 경험을 커리어 자산으로 변환하는 방법 부재

## 👥 타겟 페르소나

### 1. 김민지 (29세) - 불안한 이직 준비자
- **상황**: 더 나은 기회를 위해 퇴사 후 이직 준비 중
- **고민**: "6개월 공백기를 어떻게 설명하지?"
- **니즈**: 전환 기간을 성장 스토리로 만들기

### 2. 이준호 (32세) - 도전하는 창업가
- **상황**: 개발자 퇴사 후 스타트업 준비 중
- **고민**: "실패하면 이력에 구멍이 생길텐데..."
- **니즈**: 창업 과정의 모든 경험을 역량으로 전환

### 3. 박서연 (26세) - 재도전 수험생
- **상황**: 2년째 로스쿨 시험 준비 중
- **고민**: "또 떨어지면 2년이 날아가는 건데..."
- **니즈**: 준비 과정에서 얻은 역량 증명

## 💡 핵심 기능 설계

### 1. Smart Diary (스마트 일기)
**목적**: 부담 없이 시작할 수 있는 일기 작성

**구현 방식**:
- 3가지 가이드 질문 제공
- 음성/텍스트/사진 입력 지원
- 2-3분 내 완료 가능
- 자동 저장으로 중단 걱정 없음

**예시 질문**:
1. "오늘 가장 집중했던 일은 무엇인가요?"
2. "어떤 어려움이 있었고, 어떻게 해결했나요?"
3. "오늘 새롭게 배운 것이 있다면?"

### 2. AI Insight Engine
**목적**: 사용자가 놓치는 자신의 강점과 성장 발견

**분석 차원**:
- **스킬 추출**: 기술적/소프트 스킬 식별
- **패턴 인식**: 반복되는 강점과 성장 영역
- **컨텍스트 이해**: 상황별 대처 능력
- **진전도 추적**: 시간에 따른 발전 과정

**피드백 예시**:
```
"오늘 회의에서 팀원 간 의견 충돌을 중재하신 것이 인상적이에요. 
이런 갈등 조정 능력은 리더십의 핵심 역량입니다. 
지난 2주간 3번의 유사한 상황을 성공적으로 해결하셨네요."
```

### 3. Growth Portfolio
**목적**: 실제 활용 가능한 커리어 자산 생성

**산출물**:
- STAR 방식 경험 정리
- LinkedIn 프로필 최적화 문구
- 면접 대비 스토리 뱅크
- 역량 기반 자기소개서

## 🏗 기술적 의사결정

### 1. 왜 React + Next.js?
- **SEO 최적화**: 콘텐츠 마케팅 전략에 필수
- **성능**: SSR/ISR로 빠른 초기 로딩
- **개발 생산성**: 풍부한 생태계와 컴포넌트 재사용

### 2. 왜 PostgreSQL?
- **복잡한 관계**: 사용자-일기-인사이트-포트폴리오 간 관계
- **JSON 지원**: AI 응답 데이터 저장에 유리
- **확장성**: 향후 분석 기능 확대 대비

### 3. 왜 OpenAI API?
- **한국어 성능**: GPT-4의 우수한 한국어 이해력
- **프롬프트 유연성**: 빠른 실험과 개선
- **비용 효율**: 초기 단계에 적합한 가격
